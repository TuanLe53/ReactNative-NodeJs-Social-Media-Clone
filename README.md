# Social Media
A full-stack Social Media app built with ExpressJs, React Native and PostgreSQL.

## Screenshots


###### 

## Built With
- Frontend: 
  - React Native
  - Expo
  - Zustand
  - React Native Element
  - React Navigation
  - Socket.IO-client

- Backend:
  - NodeJs
  - ExpressJs
  - PostgreSQL
  - Socket.IO

## Features
- JWT Authentication and Authorization
- Real-time Chat
- Notification System
- Profile Customization
- Multimedia Posting
- Interactive Commenting
- Like Functionality
- Follow Capability
- User Search
